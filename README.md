# ExportJson-for-Excel-2014
Excel 插件 导出 Json 数据格式

导出文件的时候不能使用UTF8格式的文件，因为这种文件的头信息是占用字符的

Excel格式如下：

第一行：描述

第二行：类型  I->int F->float S->String B->bool

第三行：字段名
